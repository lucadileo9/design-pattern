# Glossario dei Design Pattern

> Terminologia comune utilizzata nel contesto dei design pattern

## Termini Generali

### Design Pattern
Un modello riutilizzabile per risolvere problemi comuni nella progettazione software.

### Gang of Four (GoF)
I quattro autori del libro "Design Patterns: Elements of Reusable Object-Oriented Software" (1994).

### Creational Pattern
Pattern che si occupa della creazione di oggetti.

### Structural Pattern
Pattern che si occupa della composizione di classi e oggetti.

### Behavioral Pattern
Pattern che si occupa della comunicazione tra oggetti.
